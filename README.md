# 🚀 Next Offer

## AI-Powered Placement Intelligence Platform

Next Offer is an AI-powered placement assistance platform that helps students evaluate their placement readiness through Machine Learning, Resume Analysis, Career Roadmaps, Placement Prediction, and an AI Career Chatbot.

The platform provides an interactive experience for students to analyze their profiles, identify improvement areas, and prepare effectively for campus placements.

---

# ✨ Features

### 👤 Student Profile
- Create and save student profiles
- Persistent user data using Streamlit Session State
- Placement readiness evaluation

### 📈 Placement Predictor
- Predict placement chances using Machine Learning
- Analyze student profile
- Estimate placement probability

### 📄 Resume Analyzer
- Upload Resume (PDF)
- ATS Score Analysis
- Resume quality evaluation
- Skill gap identification

### 📋 Placement Recommendations
- Company eligibility checker
- Required skills analysis
- Profile-based recommendations

### 🧠 AI Career Roadmap
- Personalized learning roadmap
- Role-specific preparation guidance
- Career improvement suggestions

### 🤖 AI Career Chatbot
Powered by Groq Llama 3.3 70B

The chatbot provides guidance regarding:

- Placements
- Resume Building
- Interview Preparation
- DSA
- Machine Learning
- Data Science
- AI
- Career Guidance
- Internship Preparation

### 📊 Analytics Dashboard

Interactive dashboard with:

- Registered Students
- Placement Readiness
- ATS Statistics
- Placement Analytics
- Branch Distribution
